# Actividad 1 - Ejercicios Prácticos

**Descripción:**  
Instrucciones para la entrega de los 10 ejercicios prácticos trabajados en clase.

**Objetivo:**  
Integrar todos los ejercicios de la actividad 1 del tema 2 en un solo repositorio de GitHub.

## Estructura del Repositorio
los archivos del proyecto se han organizado de la siguiente manera:
- **`Actividad 1/`**: Carpeta principal que contiene todos los recursos de los 10 ejercicios.
  - **`HTML/`**: Códigos fuente de los 10 ejercicios.
  - **`CSS/` y `JS/`**: Estilos y scripts correspondientes.
  - **`IMG/`**: Imágenes de prueba utilizadas.
- **`index.html`**: Archivo raíz (menú) que sirve como índice para acceder fácilmente a cada ejercicio en GitHub Pages.
- **`DOCUMENTACION/`**: Bitácora de errores encontrados durante el desarrollo y su solución.

### Ejercicio 1
Nuestro clásico "Hola Mundo", pero adornado un poco para confirmar que Bootstrap (la herramienta de diseño) sí estaba funcionando.

### Ejercicio 2
Jugamos con los textos. Aquí aplicamos cosas básicas como poner letras en negritas, subrayados y diferentes tamaños.

### Ejercicio 3
Aprendimos a hacer enlaces o "links" para que al darle clic a una palabra nos lleve a otra página de internet.

### Ejercicio 4
Le metimos imágenes a la página para que no se vea tan aburrida (usamos un león y un paisaje).

### Ejercicio 5
Un resumen de todo lo anterior. Juntamos textos, enlaces e imágenes para armar una página completa sobre "Mi página web favorita".

### Ejercicio 6
Aquí nos pusimos un poco más técnicos y armamos la "columna vertebral" de una página web moderna usando las cajas invisibles correctas (encabezado, cuerpo y pie de página).

### Ejercicio 7
Hicimos listas (como viñetas). Para hacerlo divertido, armamos la receta de los tacos al pastor.

### Ejercicio 8
Creamos una tabla cuadriculada. El ejemplo que usamos fue armar un horario de clases.

### Ejercicio 9
Armamos un formulario de contacto clásico, de esos donde pones tu nombre, correo y un mensaje, todo con el diseño bonito de Bootstrap.

### Ejercicio 10
¡El más interactivo! Hicimos una pequeña calculadora web donde escribes la temperatura en grados Celsius, aprietas un botón, y mediante código JavaScript te hace la conversión exacta a grados Fahrenheit.
